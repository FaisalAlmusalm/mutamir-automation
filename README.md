# Mutamir Automation Framework

This project is an automation testing framework for the Mutamir iOS app.

It is built using:

- Java
- Appium
- Selenium
- TestNG
- Maven

## Project Structure

```text
src/main/java/com/mutamir
├── base
├── screens
├── utils
└── listeners

src/test/java/com/mutamir/testcases
├── MainPage.java
├── OmrahPath.java
